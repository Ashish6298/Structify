Dependency Graph & Registry Design
----------------------------------
- Feature modules define their requirement chains (e.g. `rbac` -> `authentication` -> `shared-types`).
- The composition engine resolves dependencies using a topological sorting algorithm that detects circular dependencies.
- Conflicting contributions (files, scripts, environment values) are caught and rejected deterministically before files are generated.
