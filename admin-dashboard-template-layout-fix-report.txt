1. Root Cause Analysis
--------------------------------------------------------------------------------
The Admin Dashboard template's broken layout was caused by siblings (Sidebar, Topbar, KpiGrid, AnalyticsPanel, DataTable, and ProjectStatus) being output directly as child nodes of the DashboardShell flex/grid container. This forced all elements to render as horizontal columns side-by-side on desktop, compressing content into extremely narrow vertical strips. Furthermore, excessive double paddings and absolute layouts made widgets collapse, stretch, or overflow.

  - Formatted the root layout with Sidebar on the left (desktop sticky/fixed) and wrapped the rest of the children (Topbar and content widgets) in a column wrapper.
